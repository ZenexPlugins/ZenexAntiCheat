Античит с 20+ проверками

## 📦 Установка
1. Скачайте `ZenexAntiCheat.jar`
2. Поместите в папку `plugins`
3. Перезапустите сервер

## 🎯 Проверки
**Движение:** Speed, Fly, NoFall, Jesus, Phase
**Бой:** KillAura, Reach, Velocity, AutoClicker
**Интеракции:** FastPlace, FastBreak

## 🎯 Команды
| Команда | Описание | Право |
|---------|----------|-------|
| `/anticheat reload` | Перезагрузка | `zenexanticheat.admin` |
| `/anticheat info` | Информация | `zenexanticheat.admin` |
| `/anticheat alerts` | Вкл/выкл алерты | `zenexanticheat.admin` |

## 🔐 Права
| Право | Описание |
|-------|----------|
| `zenexanticheat.admin` | Административные права |
| `zenexanticheat.alerts` | Видеть предупреждения |
| `zenexanticheat.bypass` | Обход проверок |
